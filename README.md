# 简介
`wepy-plugin-iconfont`是在wepy环境中的插件。在使用wepy开发微信小程序时，可以用它方便的引入[iconfont](http://iconfont.cn/)。

# 使用
## 安装
``` bash
npm install wepy-plugin-iconfont@1.0.0
```
or
``` bash
yarn add wepy-plugin-iconfont@1.0.0
```

## 配置

wepy.config.js
``` javascript
plugins: {
  'iconfont': {
    tag: '[iconfont_tag]'
  }
}
```

`[iconfont_tag]`为下图中的值
